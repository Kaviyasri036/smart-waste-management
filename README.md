Smart Waste Management System (Django + SQLite + Bootstrap)
=========================================================

Quick start:
1. Create & activate a virtualenv (recommended).
2. pip install -r requirements.txt
3. python manage.py migrate
4. python manage.py createsuperuser
5. python manage.py runserver
6. Open http://127.0.0.1:8000/

This archive contains a minimal Django project 'smartwaste' and an app 'wasteapp'.
See comments in files for more details and possible extensions (maps, notifications, IoT).
